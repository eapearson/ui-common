This "welcome" page is designed as the exit page from KBase. But an exit page poses a dilemma -- if the page is designed as a "logged out" page, then if it is accessed in a mode other than "just having logged out" the messaging on it will be confusing. It is certainly possible to create a view or pop-up that appears immediately after logout and cannot be navigated to. At the moment, this does not seem a worthwhile pursuit.
